# fry_pan
Full featured web cloud disk based on HDFS, and Spring Boot.

## Functions
### File
- Upload
- Download
- Preview
- Trash
- Classify
- Offline download
- Folder access
### Secure
- Login/Logout
- Client transparent encryption
### Community
- Share
- Friends follow
- Friends recommend 


经试用测试

系统可用，程序会自动创建数据库表，首次使用请使用首页的注册用户进行用户注册，注册完毕后就可以注册的用户登录了

测试过可以正常上传和下载文件并且各用户之间的数据是相互独立的，算是一个比较完善的hdfs存储的演示了。
